写在前面

经过一番波折和约莫一个小时的探索，最终成功搭建起一个简易的个人网站。这个小小的网站就被命名为UncleBob的小窝啦，主要可能是作为博客但更多可能是作为树洞使用，也许会不定期更新一些文章吧。当然可能正经的文章也不会太多，更多可能是一些随笔或者一些吐槽吧。这篇文章是UncleBob的小窝的真·第一篇文章（测试用的文章已经删除了），会记录一些Q&A，也希望访问UncleBob的小窝的你能有收获（玩的开心最好啦！）。

Q&A

Q：为什么要建立这个个人网站？

一方面是受到UncleBob的挚友Zhouzi建立个人网站的启发（可在友链中找到），打算建立自己的一个小小网站用来灌水分享一些个人的小灵感或者小吐槽，另一方面约莫是觉得拥有一个个人网站确实很酷。

Q：如何建立个人网站？

UncleBob的小窝利用的是hexo这一博客框架，可以在其 官网

上了解到更多信息。GitHub Pages是 GitHub 提供的一个免费的静态网站托管服务，它允许 GitHub 用户创建和托管自己的静态网站，这些网站可以通过特定的 GitHub 仓库进行管理和托管，UncleBob的小窝就是这样的一个个人网站。具体的搭建过程参考了Bilibili的教学视频。

PS.关于为什么网站的域名（或者说我的Github用户名）中有这么奇怪的imtdof，其实是因为这名字是我随便取的（因为 Github修改用户名相当的麻烦）。

Q：打算利用这个网站做什么

UncleBob是五道口某高校学习数学与应用数学的一介书生，未来可能主要会非常不定期地更新一些数学或者其他各种方面的一些小感悟或者小吐槽。
